================================================================================
  노트북 사전 설치 — 전시 스택 실행용 (Windows 기준)
================================================================================
  exhibition-agent 레포를 돌리기 위해 한 번만 준비하면 됩니다.
  (자동 설치 아님 — 아래를 순서대로 설치·실행해야 start-dev.bat 등이 동작함)

--------------------------------------------------------------------------------
  1. Node.js (제어 웹 Next.js / npm)
--------------------------------------------------------------------------------
  목적: exhibition-agent/web 에서 npm install, npm run dev

  · 공식 사이트에서 LTS 받기: https://nodejs.org/
  · 설치 마법사 기본 옵션으로 진행 (PATH 포함 권장)

  설치 확인 — 새로 연 cmd 또는 PowerShell 에서:

       node -v
       npm -v

  버전 숫자가 나오면 성공.

--------------------------------------------------------------------------------
  2. Python (FastAPI / uvicorn)
--------------------------------------------------------------------------------
  목적: exhibition-agent 에서 pip install, uvicorn 실행

  · 공식 사이트: https://www.python.org/downloads/
  · Windows 설치 시 반드시 체크:
       [x] Add python.exe to PATH  (또는 유사 문구)

  설치 확인 — 새 터미널에서:

       python --version
       py --version

  둘 중 하나 이상이 Python 3.10 이상이면 됨.
  (어떤 PC는 python 대신 py 명령만 동작함 — 그 경우 아래에서 python 을 py 로 바꿔 쓰면 됨)

  pip 확인:

       python -m pip --version

--------------------------------------------------------------------------------
  3. 레포 클론 또는 폴더 복사
--------------------------------------------------------------------------------
  exhibition-agent 레포(폴더) 전체가 노트북에 있어야 함.

--------------------------------------------------------------------------------
  4. Python 패키지 (FastAPI 쪽)
--------------------------------------------------------------------------------
  PowerShell 또는 cmd 에서:

       cd 경로\exhibition-agent
       python -m pip install -r requirements.txt

  오류 시:

       py -m pip install -r requirements.txt

  프록시·회사망이면 방화벽 안내 필요할 수 있음.

--------------------------------------------------------------------------------
  5. Node 패키지 (제어 웹)
--------------------------------------------------------------------------------
       cd 경로\exhibition-agent\web
       npm install

  시간이 조금 걸릴 수 있음. 끝까지 에러 없이 완료되면 성공.

--------------------------------------------------------------------------------
  6. 실행 검증 (선택)
--------------------------------------------------------------------------------
  터미널 1 — 웹:

       cd 경로\exhibition-agent\web
       npm run dev

  브라우저에서 http://127.0.0.1:3001 접속 테스트 후 Ctrl+C 로 종료.

  터미널 2 — API:

       cd 경로\exhibition-agent
       python -m uvicorn app.main:app --host 0.0.0.0 --port 8000

  브라우저에서 http://127.0.0.1:8000/health → JSON 확인 후 Ctrl+C 종료.

  또는 exhibition-agent 루트에서 start-dev.bat 로 한 번에 기동 (HTTPS 제어 웹).

--------------------------------------------------------------------------------
  7. Arduino IDE (ESP 업로드만 할 때)
--------------------------------------------------------------------------------
  ESP 펌웨어 수정·업로드 시 필요. 서버 실행과는 별개.

  · https://www.arduino.cc/en/software
  · 보드 매니저에서 Espressif ESP32 패키지 추가
  · 라이브러리 매니저에서 Adafruit NeoPixel

  업로드 후 IDE를 꺼도 ESP는 단독 동작함.

--------------------------------------------------------------------------------
  자주 나는 문제
--------------------------------------------------------------------------------
  · 'node' / 'npm' 을 찾을 수 없음
       → Node 재설치 또는 터미널을 완전히 닫았다가 다시 연다.

  · 'python' 을 찾을 수 없음
       → Python 설치 시 PATH 추가 안 함 → 재설치 시 체크 또는 PATH 수동 추가.

  · pip 가 오래됨
       → python -m pip install --upgrade pip

  · npm install 중 권한·네트워크 오류
       → 관리자 권한 불필요한 경우가 많음. VPN/회사 프록시 확인.
